
# Smart Expense Tracker

Smart Expense Tracker is a Python-based application to log, analyze, and visualize personal expenses using CSV files. It allows users to add, edit, and delete expenses, view category-wise and total spending, and generate visual charts for better financial understanding. The project includes Python code in the `app/` folder, 10 screenshots of the app in the `screenshots/` folder, and a detailed PDF report in the root folder, providing a complete overview of functionality and workflow.  

**Tech Stack & Tools:** Python, CSV for data storage, Matplotlib for visualization.  

**How to Run:** Clone the repository, navigate to the `app/` folder, and run the main script `expense_tracker.py`. Follow on-screen instructions to manage expenses.  

This project is professional, recruiter-friendly, and highlights all aspects of the application, including code, visual interface via screenshots, and detailed documentation through the PDF report.
